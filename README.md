# Tailwindcss Alpinjs Simple Project
Tailwindcss Alpinjs Simple Project

## How to run project

### Install tailwindcss

`npm install -D tailwindcss`
`npx tailwindcss init`

### Generate the tailwind css tags

`npx tailwindcss -i ./src/tailwind.css -o ./src/style.css --watch`

### Install alpinejs

`npm install alpinejs`
